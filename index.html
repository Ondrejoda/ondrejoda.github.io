Skills

Python

I started programming in Python since I was 10, starting with basic TKinter applications, then trying to make more advanced projects, such as an ecosystem emulation or basic AI and then making the first "edition" of Cobra, a game engine that was made using PyGame and worked on the basis of objects.

C++

C++ was my first step up from Python. I wanted to rewrite my game engine in a compiled language, since Python is known for its low performance compared to other languages when making games. After a month of learning, I managed to make a C++ version of the Cobra game engine, which even included scenes and a more advanced particle engine.

Godot

Before I worked on the C++ version of Cobra, I was also learning how to use Godot, the open source game engine, which uses a modified version of Python for scripting. I have used it to make 11 games in total.

HTML and CSS

I have a basic understanding of HTML and CSS, since I have made some websites using it. I also know how to use SCSS and genuinely prefer it over CSS.

JavaScript

I have some experience with JavaScript, such as in web development, node.js and p5.js, although I would prefer to use other languages then JavaScript due to it's syntax.

Nim

Nim is my truly favourite programming language out there. It's syntax is very similar to Python, but it is a compiled language, so Nim's performance is much better. It can also compile itself into C, C++ and even JavaScript. Unfortunately, I am still in the process of learning how to use it properly.
